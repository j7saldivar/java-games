Instrucciones de Battleship

1) Ejecutar el BattleshipServidor.java y BattleshipCliente.java

	*Ejecucion de BattleshipServidor.java: 	java BattleshipServidor
	*Ejecucion de BattleshipCliente.java: 	java BattleshipCliente XXX.XXX.XXX.XXX
 
2) Poner las cordenadas de las naves tal como se indica en cada programa

3) Una vez ya puestas las cordenadas de las naves, esperar en el BattleshipServidor a que BattleshipCliente mande cordenadas

	*Nota: para poner las cordenadas, favor de utilizar el siguiente formato: posicion_en_x,posicion_en_y
	*Ejemplo: 4,5
	*Ejemplo: 3,4

4) Disfrutar de lo que sigue del juego
